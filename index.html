QUESTIONS & ANSWERS CSS CARDS

What is Web Development?

Web development is the process of creating and maintaining websites and web applications. It involves designing the visual elements, programming the functionality, and ensuring the site's performance and security.

What is the key difference between 'flex' and 'auto' values for flex-basis property in flexbox?

'flex' value allows flex items to expand or shrink based on available space, while 'auto' value lets items size themselves based on their content, ignoring any specified dimensions.

What is the difference between auto and 1fr in grid?

"Auto" in grid layout allows a column or row to size itself based on its content, while "1fr" assigns an equal portion of available space to the column or row inside the grid.

What is the difference between width and max-width?

"Width" sets the initial width of an element, while "max-width" limits the maximum width an element can attain, preventing it from exceeding the specified value.

What is the difference between margin and padding?

"Margin" creates space outside an element, affecting the spacing between elements, while "Padding" creates space inside an element, affecting the element's content area.

What is the difference between CSS transitions and animations?

"Transitions" define how property values change over a specified duration, while "Animations" allow for more complex, multi-step effects with keyframes.
